# Sales Analysis Dashboard using Power BI

## Project Overview
This project analyzes sales data using the Superstore dataset and presents key business insights through an interactive Power BI dashboard.

## Features
- KPI Cards
- Sales Trend Analysis
- Regional Sales Analysis
- Customer Analysis
- Product Performance
- Business Insights
- Business Recommendations

## Tools Used
- Microsoft Power BI
- Microsoft Excel
- Power Query
- DAX

## Project Structure
- Dashboard Overview
- Customer Analysis
- Business Insights
- Business Recommendations

## Outcome
The dashboard helps businesses understand sales performance, customer behavior, and product trends to support data-driven decision-making.
